# Terraform Assignments



## Assignment 1 Overview
The purpose of this project was to use Terraform to automate the deployment of a complete WordPress environment on AWS. It demonstrates how Infrastructure as Code (IaC) can be used to provision and configure cloud resources consistently, including an EC2 instance, security groups, networking, and automated software installation using user data, resulting in a fully deployed web application with minimal manual intervention. 





## Assignment 2 Overview 
This project demonstrates how to automate the deployment of an Amazon EC2 instance using Terraform and cloud-init. Terraform provisions the infrastructure, while cloud-init installs and configures NGINX during the instance's first boot, ensuring the server is fully operational without any manual setup. The project highlights Infrastructure as Code (IaC), automated instance provisioning, and a clean Terraform structure for improved maintainability and scalability.
