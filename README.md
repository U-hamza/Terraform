# Terraform Assignments



## Assignment 1 Overview
The main objective of this assignment was to deploy Wordpress using Terraform as the infracstructure. It consisted of running an EC2 instance using Wordpress user-data scripts with the correct Terraform code. 





## Assignment 2 Overview 
This assignment used Terrafrom as the infrastructure to deploy an EC2 instance with Cloud-Init. All code for the EC2 instance was configured through terraform using NGINX as the package for the cloud-init file.
